### Installation

```
yarn add react-native-unique-identifier
```
### Expo

Update app.json

```
"plugins": [["react-native-unique-identifier/plugin"]]
```
### Usage

| Function | Response | Platform |
| :-------- | :------- | :------- |
| `getiCloudRecord`      | `iCloud record ID` | iOS |
| `getAndroidAccountName`      | `google account email` | Android|
